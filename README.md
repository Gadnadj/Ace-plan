# 📦 Plan - Warehouse Management App

Application de gestion de stock avec React frontend et Node.js backend.

## 🚀 Démarrage rapide

```bash
npm install && cd backend && npm install && cd ..
npm run dev:full
```

**Frontend**: http://localhost:5173 | **Backend**: http://localhost:3001
